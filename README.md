# Terraform Scripts
My personal terraform references.

### Commands:

Steps to run:
1) Initialize and download requried plugins.
```bash
terraform init
```
2) Shows the plan of what will be created or destroyed when the config is applied. 
```bash 
terraform plan
```
3) Apply the configured resource.
```bash 
terraform apply
```
